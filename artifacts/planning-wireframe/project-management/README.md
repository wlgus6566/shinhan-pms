# 프로젝트 관리 와이어프레임

## 화면 목록

1. **index.html** - 프로젝트 목록 화면
   - 기능: 검색, 필터링(상태별), 목록 조회
   - 이동: 상세 화면, 생성 화면

2. **create.html** - 프로젝트 생성 화면
   - 기능: 신규 프로젝트 등록
   - 이동: 상세 화면(저장 후), 목록 화면(취소)

3. **detail.html** - 프로젝트 상세 화면
   - 기능: 프로젝트 정보 조회
   - 이동: 수정 화면, 목록 화면(삭제 후 또는 목록으로)

4. **edit.html** - 프로젝트 수정 화면
   - 기능: 프로젝트 정보 수정, 상태 변경
   - 이동: 상세 화면(저장 후 또는 취소)

## 화면 흐름

```
[목록] (index.html)
  ├─ [+ 새 프로젝트] → [생성] (create.html)
  │                       ├─ [생성] → [상세] (detail.html)
  │                       └─ [취소] → [목록]
  │
  └─ [카드 클릭] → [상세] (detail.html)
                      ├─ [수정] → [수정] (edit.html)
                      │              ├─ [수정] → [상세]
                      │              └─ [취소] → [상세]
                      ├─ [삭제] → [목록]
                      └─ [← 목록으로] → [목록]
```

## 로컬에서 확인하기

### 방법 1: 직접 파일 열기

브라우저에서 `index.html` 파일을 직접 열기

### 방법 2: HTTP 서버 실행 (권장)

```bash
# Python 3
cd artifacts/planning-wireframe/project-management
python3 -m http.server 8000

# 또는 Node.js (npx http-server)
npx http-server -p 8000

# 브라우저에서 열기
open http://localhost:8000
```

## 주요 기능

### index.html (목록)

- ✅ 프로젝트 카드 그리드 레이아웃
- ✅ 검색 기능 (프로젝트명)
- ✅ 필터링 기능 (상태별)
- ✅ 빈 상태 메시지
- ✅ [+ 새 프로젝트] 버튼

### create.html (생성)

- ✅ 필수 입력 필드 표시 (*)
- ✅ 유효성 검증
  - 프로젝트명: 2-100자
  - 종료일 >= 시작일
  - 중복 프로젝트명
- ✅ 에러 메시지 표시
- ✅ [생성] / [취소] 버튼

### detail.html (상세)

- ✅ 모든 프로젝트 정보 표시
- ✅ [수정] / [삭제] 버튼 (우측 상단)
- ✅ 삭제 확인 다이얼로그
- ✅ [← 목록으로] 버튼

### edit.html (수정)

- ✅ 기존 값이 미리 채워진 폼
- ✅ 상태 변경 드롭다운
- ✅ 유효성 검증
- ✅ [수정] / [취소] 버튼

## 주의사항

1. **사이드바 직접 임베드**
   - 각 HTML 파일에 사이드바가 직접 포함됨
   - fetch 방식은 로컬 환경에서 CORS 오류 발생

2. **시뮬레이션 코드**
   - JavaScript는 기능 시뮬레이션용
   - 실제 API 호출은 개발 단계에서 구현

3. **샘플 데이터**
   - 목록 화면에 샘플 프로젝트 3개 포함
   - 상세/수정 화면은 "PMS" 데이터 사용
